# 박효정 todolist

가장 기본적인 todolist의 기능을 구현하고자 했습니다.


## 필요하다고 생각한 기능들

- 입력 폼
- todolist 목록
- todolist 각 컴포넌트
- todolist 실천 여부를 판단할 체크박스
